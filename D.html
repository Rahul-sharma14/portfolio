About the Project

Digital Diary Digital-Diary is a feature-rich digital journaling web application designed to provide users with a secure, personalized space to record their daily thoughts, emotions, and experiences. Built using React for the frontend and Django for the backend, the platform emphasizes user privacy, insightful analytics, and intuitive design.

Key Features:
Daily Motivational Quotes to inspire each day.
Create Diary Entries with categories, tags, and image uploads.
Mood Tracker to record emotional states and view trends over time.
Secret Entries with encryption for enhanced privacy.
Edit, Update & Delete Entries seamlessly.
Share Entries with other users (optional).
User Authentication (Login/Signup) with profile picture upload.
Export Entries to PDF for offline access.
Dark Mode Toggle for comfortable writing at any time.
Calendar View to navigate entries by date.
Analytics Dashboard showing mood patterns, entry count, and category stats.
